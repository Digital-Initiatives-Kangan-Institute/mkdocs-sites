# CIII Programming and Design
This repository contains the mkdocs source for the Certificate III in Information Technology programming and design thinking unit activities.

## Usage
1. Clone this repository
2. Create a python environment and activate it
3. Run the install script (`install` bash script)
4. Run `./serve` to start the dev server
5. Run `./build` to generate a build for GitHub pages

## Authoring
To help you author pages, read the [AUTHORING.md](./AUTHORING.md) guide.

## Dependencies
- python / pip
- mkdocs
- mkdocs-material
- mkdocs-enumerate-headings-plugin
