2. Class Activity 2 - Double Diamond - Define

From the discoveries from Class Activity 1, you are now going to converge your findings and develop a problem statement to solve for.

From the your findings in Class Activity 1:

  • Determine the users/stakeholders
  • Group ideas and observations

Finally, create a statement to define the problem that you intend to solve. For example, "How might we utilise web-browser based games to help people develop basic computing skills?"

You will be asked to present your problem statement and users to the class.
