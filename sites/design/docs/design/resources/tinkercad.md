# TinkerCAD - 3D Modeling Tool
**Tinkercad** is a beginner-friendly tool for learning **3D modelling**. It runs in a web browser and allows users to create models by **combining basic shapes such as cubes, cylinders, and spheres**. These shapes can be **resized, rotated**, and **grouped** to **build more complex designs**, helping users understand the basic concepts of creating objects in a 3D space.

Tinkercad is also commonly used to **prepare models for 3D printing**. Once a design is complete, it can be **exported** and **imported** into 3D printing software for **printing**. This makes it a practical tool for designing simple items that can be made using a 3D printer.
